## Getting Started

first run, redis-server:

second, run the development server:

```bash
npm run dev

yarn dev
# or
pnpm dev
# or
bun dev
```

Download Redis on your local

See this video to install redis:

https://www.youtube.com/watch?v=DLKzd3bvgt8&t=1s

```bash
npm install redis
```
